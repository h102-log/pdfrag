# 냉각수 순환 펌프 P-101 운영 매뉴얼

## 1. 개요

냉각수 순환 펌프 **P-101**은 메인 냉각 루프의 핵심 설비다. P-101은 모터 **M-22**로 구동되며, 베어링 **B-7**을 통해 축을 지지한다. 펌프 출구는 열교환기 **HX-3**으로 연결된다.

## 2. 구성 요소

P-101은 임펠러, 케이싱, 기계식 실(seal) **S-5**로 구성된다. 기계식 실 S-5는 누설 발생 시 가장 먼저 점검한다.

## 3. 점검 주기

| 항목 | 점검 주기 | 담당 |
| :--- | :--- | :--- |
| 베어링 B-7 진동 | 1주 | 설비팀 |
| 기계식 실 S-5 누설 | 1일 | 운전팀 |
| 모터 M-22 절연저항 | 1개월 | 전기팀 |

## 4. 이상 시 처리 절차

베어링 B-7 진동이 기준치를 초과하면 모터 M-22를 정지하고 윤활유를 점검한다. 윤활유 점검 후에도 진동이 지속되면 베어링 B-7을 교체한다. 교체 작업은 설비팀이 수행한다.
